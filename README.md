# 🌐 Shaikh Masud – Portfolio Website

Welcome to the source code for my personal portfolio website – [**shaikhmasud.github.io**](https://shaikhmasud.github.io).
This site showcases who I am, what I’ve built, and how to get in touch with me.

---

## 🧾 About the Project

This is a responsive React-based personal portfolio site that includes:

* 👨‍💻 A professional introduction and background
* 🛠️ A detailed list of projects with links
* 🧠 Skills and technologies I work with
* 📬 Contact information

The website is deployed using **GitHub Pages**.

## 🚀 Live Site

🔗 [https://shaikhmasud.github.io](https://shaikhmasud.github.io)

## 🛠 Tech Stack

* ⚛️ **React.js** – UI library used to build the entire site
* 🎨 **CSS3 / Bootstrap** – For styling and layout
* 💡 **JavaScript (ES6+)** – Interactive and functional components
* 📦 **React Router** (if navigation is included)
* 🌐 **GitHub Pages** – Static site hosting

## 🖼 Features

* ✨ Modern, clean UI
* 🧑‍💻 About Me section with tech stack
* 📂 Portfolio/Project cards with GitHub/live links
